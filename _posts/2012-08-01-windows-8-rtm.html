Hoje a Microsoft anunciou que atingiu o milestone RTM do Windows 8 e Windows Server 2012. Isso significa que a versão final está pronta e foi disponibilizada para fabricantes de PC.

Para o público geral o Windows só estará disponível para compra a partir do dia 26 de outubro, mas quem tem assinatura MSDN ou TechNet já poderá fazer download da versão final no próximo dia 15 de agosto (em exatamente 2 semanas), junto com as versões finais das ferramentas de desenvolvimento.

Quem comprar um computador novo com Windows 7 hoje poderá fazer upgrade para o Windows 8 Pro por apenas US$ 14,99 a partir do dia 26 de outubro. Computadores comprados antes poderão fazer o upgrade por US$ 39,99.

Todas essas notícias são excelentes, tanto pela disponibilidade quanto os valores. Agora é só esperar mais um pouquinho para poder começar a desfrutar da versão final.

Eu já estou usando a consumer preview desde de o lançamento tanto no trabalho como no meu laptop pessoal e o único problema que tive até agora foi com vídeos no laptop pois os drivers ainda não estão 100% no consumer preview. Espero que isso se resolva agora com RTM. Agora é começar a me preparar para zerar essas máquinas pois daqui duas semanas vou instalar o Windows 8 em todos os computadores que uso :)
