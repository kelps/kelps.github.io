Hoje, às 13:00 (horário de Brasília) será transmitido ao vivo pelo Channel 9 via Live Streaming o Windows Phone Summit.

Não há muitos detalhes concretos sobre o conteúdo que será apresentado no evento, mas sabemos que finalmente teremos maiores informações sobre o Windows Phone 8.

Se você se interessa por desenvolvimento em XAML em geral ou simplesmente quer saber mais sobre o futuro do Windows Phone, este evento será imperdível.

Segue abaixo uma lista de rumores sobre o que talvez seja anunciado/explicado hoje:

  • Suporte a processadores com múltiplos núcleos
  • Telas HD e novas resoluções
  • Suporte a cartões de memória
  • Maior integração com Skype
  • Mais funcionalidades e personalizações no aplicativo da câmera
  • Suporte a NFC tanto para pagamentos como para compartilhamento de conteúdo e emparelhamento de dispositivos
  • Integração com Windows 8
  • IE10
  • Maior controle sobre consumo de banda de dados e monitoramento de bateria
  • Uso do mesmo núcleo do Windows 8 ou um subset dele
  • Subset das APIs e ferramentas de desenvolvimento de aplicações Metro do Windows 8 para desenvolver para Windows Phone 8
  • Possibilidade ou não de atualização dos aparelhos atuais para a nova versão do sistema operacional
