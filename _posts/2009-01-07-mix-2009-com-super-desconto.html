Galera,

Como vem ocorrendo todo ano e foi divulgado recentemente pelo René, as inscrições para o MIX 2009 (em Las Vegas) estão com desconto de US$ 200,00 até dia 15/01/2009, saindo por US$ 1.195,00, para quem se inscrever online. É um desconto significativo de mais de R$ 400,00 que vale a pena ser aproveitado.

Mas para quem for rápido, neste momento é possível se inscrever aproveitando um desconto especial e pagando "apenas" US$ 795,00. Pelo que eu vi no site este desconto é apenas para os primeiras 200 pessoas a utilizar o código!

Para utilizar o desconto será necessário seguir os passos abaixo:

  • Abrir a página de incrição online e verificar se a promoção ainda está valendo (procurar pelo texto "Limited Offer: Save 40% on Conference Registration..." logo abaixo do título EVENT PRICING)
  • Clicar no Register Now;
  • Escolher a opção Register for the event using an RSVP Code
  • Digitar o seu email, escolher uma senha e informar o código da promoção MIXspecial1.
  • Seguir o passo a passo do cadastro até o final e efetuar o pagamento.

Eu já garanti o meu desconto.
