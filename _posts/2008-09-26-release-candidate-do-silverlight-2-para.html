Faz tempo que não posto nada por aqui mas hoje eu não podia deixar passar pois foi lançado o primeiro release cadidate do Silverlight 2.

Mas a versão que foi disponibilizada hoje é apenas a developer e não tem licença Go-Live. O objetivo desse lançamento é dar tempo para os desenvolvedores portarem suas aplicações da versão Beta 2 para a versão final. Isso é muito importante pois há quebra de compatibilidade entre o Beta 2 e o RC. De acordo com o post publicado hoje por Scott Guthrie, a versão final está bem próxima do lançamento.

Entre as novidades estão os tão esperados controles ComboBox, PasswordBox e ProgressBar. Além disso, o skin básico dos controles está mais refinado e ainda deve mudar mais um pouco até a versão final, o que dará uma aparencia mais profissional para as aplicações que os utilizarem sem customização. No post do Scott Guthrie é possível ver como estão ficando os controles DataGrid, RadioButton, CheckBox e DatePicker.

Este Release funciona com o Microsoft Visual Web Developer 2008 Express Edition.

Também haverá novos controles a ser lançados à parte do runtime principal, mas não há mais detalhes sobre isso no momento.

Foi lançado hoje também um service pack para o Expression Blend 2 com suporte a esse novo release do Silverlight.
